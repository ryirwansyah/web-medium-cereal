# KKSTNIAD 2021 - Cereal

## How to start the challenge : 
### With Docker 
1. docker-compose up --build
2. access port 20003
3. you good to go

### Run with npm
1. npm install
2. npm start
3. it should be running on port 3000 (default)
4. you good to go

PS. if u wanna check the solution just move to the `solution` branch.
